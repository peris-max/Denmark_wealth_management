# 📊 Overview

FormueOS is a **React-based portfolio dashboard** that explores how financial advisors could manage client portfolios through a clean and intuitive interface.

The dashboard provides a high-level overview of client portfolios, including:

- portfolio value
- monthly returns
- diversification insights
- risk alerts
- portfolio health scoring

The project focuses on **clarity, usability, and modern financial UX design**.

---

# ✨ Key Features

### Client Portfolio Dashboard
View each client's portfolio value, performance metrics, and portfolio health score.

### Asset Allocation Insights
Visualise diversification across different asset classes.

### Portfolio Health Score
A simplified score representing diversification quality and portfolio balance.

### Risk Alerts
Identify potential issues such as:

- bond duration risk
- inflation exposure
- asset concentration

### Advisor Workflow
Switch easily between multiple clients and review portfolio details.

### Firm-Level Overview
Track total assets under management across all clients.

# 💡 Project Motivation

Wealth management platforms are often complex and difficult to navigate.

FormueOS explores how a modern fintech interface could simplify advisor workflows by focusing on:

- clearer portfolio insights
- better visualisation of diversification
- intuitive risk alerts
- clean financial data presentation

The project demonstrates how **modern frontend technologies can improve financial advisory tools**.
